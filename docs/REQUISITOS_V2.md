# Requisitos V2 — IRREAL App Cloud

## Controle geral

Usuário: Kleber Nunes Silva.

Permissões:
- cadastrar professores;
- desativar professores;
- cadastrar cursos;
- cadastrar turmas;
- atribuir professor a turma;
- cadastrar alunos em qualquer turma;
- desativar alunos;
- visualizar todas as turmas;
- visualizar todos os entregáveis;
- exportar dados;
- alterar senhas.

## Professor

Permissões:
- visualizar apenas suas turmas;
- cadastrar alunos nas turmas sob sua responsabilidade;
- desativar vínculo do aluno com sua turma;
- criar missões;
- registrar liderança;
- lançar/debitar IRREAIS;
- criar itens da loja;
- visualizar entregáveis dos alunos de suas turmas.

## Aluno

Acesso:
- nome completo;
- senha.

Permissões:
- ver suas turmas;
- ver saldo;
- consultar missões;
- enviar entregáveis;
- consultar extrato.

## Entregáveis

Cada entregável contém:
- turma;
- aluno;
- missão;
- título;
- descrição;
- link externo;
- arquivo opcional;
- e-mail do professor responsável;
- status de envio.
